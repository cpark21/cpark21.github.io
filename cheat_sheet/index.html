How to set up an HTML Document

Before you even start writing your code, you need to set up your HTML file.
Let's see how we can do that.

The Basics


Naming your HTML

For any website, it is the industry standard to name your main html file "index.html." Now, when other people are looking through your files, they will know that your "index.html" is your main html file!


Declaring Document Type (DOCTYPE)

Begin every HTML document with <!DOCTYPE html>. If you inspect any website on your internet browser by pressing F12, you will notice that the first line is always <!DOCTYPE html>. This is to declare that the file is an HTML file for programs to read.

The Head


The head wraps around text that sets up your html document. Although nothing inside the <head> actually appears on your website, it is still crucial.


The <title>

This is the the text that will appear as the tab name. For this website, it is "HTML Basics - Cheat Sheet" as shown above.


Linking files

You can link files using the <link> For this website, the <link> tag is used to link the .css and font to the html file.

The Body

The "meat" of your website. ALL your content will be included in the body. This includes the tags such as the header, main and footer.

Some Useful HTML Tags


Tag Function
<a> Hyperlinks text
<h1-h6> Headers with different sizes
<span> Classify specific text
<table> Wrapper element for tables
<ul/ol> Unordered/ordered list
<button> Makes text a button
